Gerando notebook
----------------------

Requerimentos: [Python 2/3](https://www.python.org/), [latexmk](https://www.ctan.org/pkg/latexmk/)

```bash
sudo apt install python3
```

```bash
sudo apt install latexmk
```

Comando:

```bash
python3 notebookcreator.py
```
